Install elasticsearch:

https://www.elastic.co/guide/en/elasticsearch/reference/current/zip-windows.html

follow windows install instructions

optional: add --> ingest.geoip.downloader.enabled : false 
into elasticsearch.yml in the config folder to help save memory. do this before running the .bat to start elastic

once elastic is run, save the auto generated password for user elastic into config.py as

elastic_pass = "<autogenerated password>"


python -m pip install elasticsearch

^ use to install elasticsearch python library
